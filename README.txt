Overview:
--------
This module allows users to generate timelines based on the contents of 
your Drupal site. This visualization is based on content date fields assigned
to content types on your site.

Users can also save generated timelines to easier find them later. 

Installation:
------------
1. Place this module directory in your modules folder (this will
   usually be "sites/all/modules/").
2. Go to Administer -> Site Building -> Modules and enable the module.

Please note the following dependencies for this module:
  * cck
  * timeline

Configuration:
-------------
You can configure the usergen_timeline module by going to Administer -> Content 
Management -> User Generated Timelines.
